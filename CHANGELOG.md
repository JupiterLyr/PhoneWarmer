# Changelog

---

## [1.3.4]
### Added
- 加入了电池电压的显示

### Changed
- 去除了瞬时功率显示数值中的逗号
- 修改了部分 UI 的文字和配色
- 调整了软件依赖包

## [1.3.2]
### Added
- 添加了内存占用量的获取与呈现逻辑
- 加入了 CPU 频率、电池电流、电池瞬时功率、系统热状态的获取与呈现

### Changed
- 将界面显示刷新逻辑从 `MainActivity` 独立到 `MainViewBinder`
- 将强度上限调整至 9

### Fixed
- 修改了 CPU 负荷获取与呈现方式，现在部分机型支持获取系统负荷，Android 12+ 机型只能获取当前进程负荷，此处做了兼容性处理
- 修复了深色模式的显示问题，修正了部分 UI 文字语义

## [1.2.1]
### Changed
- 优化了部分 UI 设计，如组件卡片显示、电量分色提醒等

### Fixed
- 修复了 GPU 设备未绑定、GLSL 精度不匹配等问题
- 调整了 CPU 利用率获取方式，并修复了始终显示 0% 的问题
- 修复了 GPU 渲染可视化窗口始终全黑的问题

## [1.1.0]
### Added
- 加入了 GPU 引擎，并在 `WorkloadEngine` 中集成了 GPU 运算功能
- 添加了系统状态监视器模块

### Changed
- 提升了 CPU 运算的复杂度

### Fixed
- 修复了部分 UI

## [1.0.0]
- 基本功能实现，初步确定 UI 设计，但仍存在一些 bug
- 未加入 CPU、GPU 监视器，且 GPU 尚未参与运作
